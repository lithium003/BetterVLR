# BetterVLR

VLR.GG but better!

### Features:
- Quicklinks to each year's VCT page and the transfer hub
- Ability to filter matches and events on the homepage by region (Americas, EMEA, Pacific, China) and competition level (VCT, VCL, GC)
- Map winrate percentages listed on match pages
- Past players listed on team pages
- Trophies listed on player and team pages

## Local Installation (Chrome)

Prerequisites
- Node.js (16+ recommended)
- npm or yarn

Build (recommended)
1. Install dependencies:

```powershell
npm install
```

2. Build the extension (production bundle):

```powershell
npm run build
```

3. Load unpacked in Chrome:

- Go to `chrome://extensions/`
- Enable Developer mode
- Hit `Load unpacked`
- Select the newly created `build` folder created in step 2
